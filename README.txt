CleanSuite - Frontend React (Evidencia GA7-220501096-AA4-EV03)
_______________________________________________________________

Este proyecto corresponde al componente frontend del proyecto formativo CleanSuite,
desarrollado con React JS, de acuerdo con la evidencia GA7-220501096-AA4-EV03 del SENA.

Módulos implementados en el frontend:
- Inicio de sesión y registro.
- Home de usuario (información básica y accesos).
- Bandeja de tareas (listado y filtro por ID/nombre).
- Gestión de tarea (checklist, estados y observaciones).
- Asignar tareas (solo rol ADMIN).

Tecnologías:
- React 18
- React Router DOM 6
- Webpack + Babel para empaquetado

Scripts (una vez instalado Node y las dependencias con `npm install`):
- `npm start`  -> Levanta servidor de desarrollo en http://localhost:3000
- `npm run build` -> Genera build de producción en carpeta /dist
